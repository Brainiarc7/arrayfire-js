# Seq class

## constructors

- `new Seq(begin, end)`
- `new Seq(begin, end, step)`

**Arguments**:

- **begin: Number** - begin of sequence values
- **end: Number** - end of sequence values
- **step: Number** - step size (default is 1)

## Properties

### begin

**Value:** begin of sequence values

### end

**Value:** end of sequence values

### step

**Value:** step size
